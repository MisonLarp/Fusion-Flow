🚀 Setup Guide
1. Install Python
- Make sure that no other python version is installed
- Open the file python-3.10.11.exe
- IMPORTANT: Tick the box that says “Add Python to PATH”
- Click through the installer
- Run the "Dependencies.bat" file

  
2. Open DaVinci Resolve
- Start DaVinci Resolve
- Open any project
- Go to the Fusion page

3. Start the Tool
- Run FFLOW.exe
- Click the ⚙️ Settings
- Go to the “General” tab
- Click “Default Path”
- This should automatically find your DaVinci Resolve Developer Folder
- If it doesn’t: manually select your DaVinci Resolve folder
- Close the settings window

4. Learn the Tool
- A tutorial is already built into the app
- Or watch Clout’s tutorial if you prefer video

⚠️ Common Mistakes (read this before you complain)
- Didn’t tick “Add Python to PATH” → it won’t work
- Didn’t open Fusion page → it won’t connect
- Wrong DaVinci path → also won’t work
